# Frontend Development - Hands_On_1

## Module
Frontend Development

## Hands-On Title
HTML5 Semantic Structure & CSS3 Foundations

---

## Objective

The objective of this hands-on is to understand the fundamentals of HTML5 semantic elements and CSS3 styling by developing the basic structure of a Student Portal web page. The exercise focuses on creating a well-structured, accessible webpage and applying CSS fundamentals such as selectors, typography, spacing, the box model, and basic layouts.

---

# Task 1 - Build the Page Skeleton with Semantic HTML5

## Objective

Create a semantic and accessible HTML5 webpage using the appropriate structural elements.

## Features Implemented

- Created a valid HTML5 document.
- Added the following meta tags:
  - UTF-8 Character Encoding
  - Responsive Viewport
- Added an appropriate page title.
- Designed the webpage using semantic HTML5 elements:
  - `<header>`
  - `<nav>`
  - `<main>`
  - `<section>`
  - `<article>`
  - `<footer>`
- Added a navigation menu containing:
  - Home
  - Courses
  - Profile
  - Grades
- Created a Hero section containing:
  - Welcome heading
  - Description
  - Explore Courses button
- Created a Courses section containing three course cards using `<article>`.
- Added a footer with copyright information.
- Validated the HTML using the W3C HTML Validator.

---

# Task 2 - Apply CSS3 Fundamentals

## Objective

Style the Student Portal webpage using CSS3 fundamentals.

## Features Implemented

- Linked an external stylesheet (`styles.css`).
- Applied a CSS Reset using the universal selector.
- Styled the body with:
  - System font
  - Background color
  - Default text color
- Designed the header using Flexbox.
- Styled the navigation links.
- Added spacing and typography improvements.
- Styled the Hero section.
- Designed the Explore Courses button with a hover effect.
- Created course cards using a custom `.course-card` class.
- Applied:
  - Border
  - Border Radius
  - Box Shadow
  - Padding
- Styled the footer.
