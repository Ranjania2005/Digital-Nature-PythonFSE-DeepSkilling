# Frontend Development - Hands_On_2

## Module

Frontend Development

## Hands-On Title

CSS Flexbox, CSS Grid & Responsive Web Design

---

## Objective

The objective of this hands-on is to build a responsive Student Portal web page by applying modern CSS layout techniques. The exercise focuses on creating flexible layouts using Flexbox, designing responsive card layouts with CSS Grid, and implementing responsive web design using media queries.

---

# Task 1 - Flexbox Layout

## Objective

Create flexible page layouts using CSS Flexbox.

## Features Implemented

- Designed a responsive header using Flexbox.
- Aligned the website title and navigation horizontally.
- Created a responsive navigation menu.
- Designed the Hero section using Flexbox.
- Added a Student Statistics section displaying:
  - Courses Enrolled
  - GPA
  - Semester
- Styled the statistics cards using Flexbox.
- Maintained the course section from Hands-On 1.

---

# Task 2 - CSS Grid Layout

## Objective

Implement CSS Grid to create a responsive course layout.

## Features Implemented

- Created a CSS Grid container for course cards.
- Displayed multiple course cards using Grid.
- Added additional course cards to improve layout.
- Applied spacing between grid items using `gap`.
- Designed responsive cards with:
  - Border
  - Border Radius
  - Box Shadow
  - Padding
- Used `repeat()` and `minmax()` for responsive grid columns.

---

# Task 3 - Responsive Web Design

## Objective

Build a responsive webpage that adapts to different screen sizes using Media Queries.

## Features Implemented

- Implemented a Mobile-First design approach.
- Added responsive typography using `clamp()`.
- Used viewport height (`vh`) for flexible Hero section sizing.
- Created responsive layouts using Media Queries.
- Implemented breakpoints for:
  - Mobile Devices
  - Tablets
  - Desktop Screens
- Updated navigation layout based on screen width.
- Adjusted the statistics section for different devices.
- Configured the course grid layout:
  - Mobile в†’ 1 Column
  - Tablet в†’ 2 Columns
  - Desktop в†’ 3 Columns
