# Python Backend Framework - Hands On 1

## Task 1 - Django Project Setup

### Objective
Set up a Django development environment and create a Django project with an application.

### Steps Performed
- Created a Python virtual environment
- Activated the virtual environment
- Installed Django
- Created a Django project named `coursemanagement`
- Created a Django application named `course`
- Added the application to `INSTALLED_APPS`
- Applied the default migrations
- Started the Django development server

### Expected Output
The Django default welcome page is displayed successfully.
